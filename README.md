LeiloesTDSat — Sistema de Leilões


Sobre o Projeto
O LeiloesTDSat é um sistema informatizado desenvolvido para automatizar e organizar as rotinas de uma casa de leilões. O sistema permite o cadastro de lotes, controle de lances, gerenciamento de clientes e acompanhamento dos resultados dos leilões, proporcionando mais eficiência e rastreabilidade nos processos da empresa.

Tecnologias Utilizadas
